# NetJSON
## Overview
The Library that loads JSON files from the internet.
## Docs
### Usage
#### `loadnetjson`

Devription: Loads a JSON file from a URL

Arguments/Input: `loadnetjson(<URL>)`

Argument/Input Info: `<URL>` needs to be in format `http://example.com/file.json`
### Requirements
 - urllib3
### pslib module
`pslib.netjson`
